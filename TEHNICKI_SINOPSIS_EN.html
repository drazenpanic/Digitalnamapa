TECHNICAL SYNOPSIS - Digital Map of Sremska Mitrovica

Report date: 2025 | Version: 1.0 | Status: Production ready

PROJECT OVERVIEW

Digital Map is a web application developed for the Lux Aurelius 2025 Festival in Sremska Mitrovica. The application provides an interactive digital city map with points of interest (POI), event information, and navigation features.

TECHNOLOGY STACK

Backend

  • .NET 8.0 - ASP.NET Core MVC application
  • C# - main programming language
  • MVC Pattern - Model-View-Controller architecture

Frontend

  • Bootstrap 5 - responsive UI framework
  • Leaflet.js - interactive map library
  • JavaScript (ES6+) - client-side logic
  • HTML5/CSS3 - markup and styling

Localization

  • ASP.NET Core Localization - multi-language support
  • RESX files - resources for Serbian (sr) and English (en)

Deployment

  • Docker - application containerization
  • Multi-stage Dockerfile - optimized production build

KEY FEATURES

1. Interactive Digital Map

  • Custom map - hand-drawn city map used as an image overlay
  • Leaflet.js integration - advanced map interaction (zoom, pan, click)
  • Coordinate system - transformation between geographic coordinates (lat/lng) and map pixel coordinates
  • Barycentric interpolation - precise transformation using reference points

2. Points of Interest (POI)

  • 20+ POI locations implemented:
    • Landmarks (Imperial Palace, City Park, Museum of Srem, Basilica, etc.)
    • Services (WC, Parking, Water, First Aid, Food)
    • Events (Roman Camp, Roman Bazaar, Žitni Square, etc.)
  • Custom icons - each POI has its own unique icon
  • Informational popups - display name, description, and events on click
  • Detail pages - dedicated pages for key POIs with full information

3. Geolocation

  • User position tracking - real-time display of user location on the map
  • GPS integration - uses the HTML5 Geolocation API
  • Visual indicators - blue marker with accuracy circle
  • Corner marker - indicator when the user is outside the visible map area
  • Toggle button - enable/disable location tracking

4. POI Filtering

  • Categories:
    • Landmarks
    • Events
    • Services
  • Dynamic filtering - show/hide markers based on category
  • Active filter indicator - visual indicator of the active filter

5. Events and Schedule

  • Events linked to POIs - each POI can have multiple events
  • Event details - date, time, description
  • Map display - events shown in map popups
  • Schedule page - dedicated page for viewing the program (prepared)

6. Bilingual Support

  • Serbian and English - full support for both languages
  • Cookie-based localization - stores user language preference
  • RESX resources - centralized translation management
  • Dynamic switching - change language without reloading the page

7. Google Maps Integration

  • Direct links - each POI has a link to Google Maps
  • Coordinate precision - accurate geographic coordinates for navigation

8. Responsive Design

  • Mobile-first approach - optimized for mobile devices
  • Adaptive layout - adjusts to different screen sizes
  • Touch-friendly - support for touch gestures on mobile devices

APPLICATION ARCHITECTURE

Models

  • PointOfInterest - main model for points of interest
    • Basic data (name, description, coordinates, type)
    • Events - list of related events
    • Status (IsActive) - activation/deactivation of POIs
  • Event - event model
    • Title, date, time, description

Controllers

  • POIController - POI management
    • Map() - main page with the map
    • Details(int id) - detailed POI page
    • Program() - schedule page
  • HomeController - base controller
    • SetLanguage() - language switching

Views

  • Map.cshtml - main interactive map
  • Details.cshtml - detailed POI page with:
    • Full location description
    • List of events
    • Integrated Leaflet map
    • Google Maps link
  • _Layout.cshtml - main layout with navigation and language switcher

JavaScript

  • mapa.js - main map logic
    • Leaflet map initialization
    • Adding markers
    • Geolocation
    • Filtering
    • Coordinate transformation

LOCALIZATION

Implementation

  • IStringLocalizer<SharedResource> - dependency injection for localization
  • RESX files:
    • SharedResource.sr.resx - Serbian translations
    • SharedResource.en.resx - English translations
    • Views.Shared._Layout.sr.resx - layout translations (Serbian)
    • Views.Shared._Layout.en.resx - layout translations (English)

Coverage

  • All UI elements are localized
  • Map popups
  • Navigation
  • Detail pages
  • Messages and labels

MAP FUNCTIONALITY

Coordinate System

  • Image Overlay - custom map image is used
  • Simple CRS - Leaflet Simple Coordinate Reference System
  • Transformation - barycentric interpolation between 3 reference points:
    • City Park
    • Žitni Square
    • City Library

Marker Management

  • Custom icons - each POI has its own PNG icon
  • Layer groups - markers organized into groups
  • Popup content - dynamically generated popups with localized text
  • Legend - interactive legend with clickable elements

Interactivity

  • Zoom controls - built-in Leaflet controls
  • Pan - move the map with mouse/touch
  • Click handlers - clicking a marker opens a popup
  • Legend click - clicking the legend centers the map on the POI
