PREDLOG PROJEKTA - Digitalna Razglednica

PREGLED PROJEKTA

Digitalna Razglednica je web aplikacija koja omogućava posetiocima Festivala Lux Aurelius 2025 da kreiraju i šalju digitalne razglednice sa pozdravom iz Sremske Mitrovice. Aplikacija kombinuje rimsku estetiku sa modernom tehnologijom, omogućavajući korisnicima da pošalju personalizovane razglednice preko email-a.

FUNKCIONALNOSTI

1. Frontend - Razglednica Interface

  • Vizuelni dizajn razglednice - ilustrovana razglednica sa rimskim motivima
    • Ilustracija Sremske Mitrovice sa rimskim elementima
    • Zlatne boje (#D3B865) i tamne pozadine (#211B0D)
    • Rimski fontovi (Marcellus, Amatic SC)
    • Dekorativni elementi (banneri, simboli, itd.)
  • Forma za unos podataka (desna strana):
    • Email input polje (validacija)
    • Textarea za kratku poruku (maksimalno 200-300 karaktera)
    • Dugme "Pošalji razglednicu"
    • Loading indikator tokom slanja
    • Success/Error poruke
  • Preview razglednice (leva strana):
    • Real-time prikaz kako će razglednica izgledati
    • Prikaz unetog teksta na razglednici
    • Responsive dizajn

2. Backend - Email Slanje

  • Email servis - slanje HTML email-a sa razglednicom
  • Email template - HTML template sa stilizovanom razglednicom
  • Validacija - validacija email adrese i teksta
  • Rate limiting - zaštita od spam-a (npr. max 3 razglednice po IP-u u sat vremena)
  • Logging - logovanje svih poslatih razglednica

3. Dizajn i UX

  • Rimski estetika - konzistentna sa Digitalnom Mapom
  • Animacije - smooth animacije pri slanju
  • Responsive - optimizovano za mobile i desktop
  • Dvijezičnost - srpski i engleski jezik

TEHNOLOGIJSKI STEK

Backend

  • .NET 8.0 - ASP.NET Core MVC
  • C# - glavni programski jezik
  • MailKit ili SmtpClient - za slanje email-a
  • FluentValidation (opciono) - za validaciju

Frontend

  • Bootstrap 5 - responsive framework
  • JavaScript (ES6+) - interakcija i validacija
  • HTML5/CSS3 - markup i stilizacija
  • Google Fonts - Marcellus, Amatic SC, Open Sans

Email

  • SMTP Server - konfiguracija za slanje email-a
    • Opcije: Gmail SMTP, SendGrid, Mailgun, ili lokalni SMTP
  • HTML Email Template - responsive email template

Lokalizacija

  • ASP.NET Core Localization - ista kao Digitalna Mapa
  • RESX fajlovi - srpski i engleski prevodi

Resursi

CSS/JS

SIGURNOST

1. Validacija

  • Email format validacija
  • XSS zaštita (HTML encoding)
  • CSRF token zaštita
  • Input length ograničenja

2. Rate Limiting

  • Ograničenje broja razglednica po IP adresi
  • Implementacija pomoću IMemoryCache
  • Logovanje pokušaja

3. Email Validacija

  • Verifikacija email adrese pre slanja
  • Blacklist za spam email-ove (opciono)
